GitHub上傳說明
===============

1. 使用方法：
   - 雙擊 upload_to_github.bat 執行
   - 系統會自動添加所有檔案到Git
   - 輸入提交訊息（可選，按Enter使用預設訊息）
   - 自動推送到GitHub

2. 注意事項：
   - config.py（包含API密鑰）不會被上傳
   - logs/目錄下的日誌檔案不會被上傳
   - trade_history.json（交易記錄）不會被上傳
   - 其他敏感檔案也已經被忽略

3. GitHub地址：
   https://github.com/billshiou/funding_rates_bn_r1

4. 如果遇到問題：
   - 確保你已經設置好Git和GitHub
   - 確保你有推送權限
   - 如果是首次推送，可能需要設置Git用戶名和郵箱：
     git config --global user.name "你的用戶名"
     git config --global user.email "你的郵箱"

5. 手動上傳命令（如果腳本有問題）：
   git add .
   git commit -m "更新資金費率套利機器人"
   git push origin main
